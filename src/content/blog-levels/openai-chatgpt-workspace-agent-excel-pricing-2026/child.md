OpenAI は 2026年7月6日、ChatGPT Business で Workspace Agents と ChatGPT for Excel/Sheets の料金を token-based pricing にすると発表しました。簡単に言うと、何回使ったかだけではなく、AI が読んだ文字量、前に読んだ内容を再利用できた量、AI が出した答えの長さによって credits が変わる仕組みです。

これは、ChatGPT を会社で使うときにかなり重要です。今までは「席を何人分買うか」が中心でした。でも、業務エージェントや表計算AIが広がると、「どの部署が、どんな仕事で、どれくらい credits を使うか」も見なければなりません。

## 何が課金対象になったのか

今回の対象は、Workspace Agent runs と ChatGPT for Excel/Sheets tasks です。Workspace Agents は、会社の中で繰り返す仕事をエージェントとして作り、アプリやファイルとつないで動かす機能です。Excel/Sheets tasks は、表計算の中でデータ整理、説明、更新、分析を助ける機能です。

OpenAI の rate card では、Excel/Sheets の典型的な task は GPT-5.5 で 5 から 20 credits、Workspace Agent run は 5 から 25 credits 程度になることがあると説明されています。ただし、これは目安です。大きなファイルを読ませたり、長い資料を作らせたりすると増えます。

PowerPoint も関係します。Business と Enterprise では、ChatGPT for PowerPoint は 2026年8月6日まで無料です。その後は、Excel/Sheets と同じ token-based pricing に入ります。つまり、無料期間に使い方を試し、課金後にどれくらい費用が出そうかを見ておく必要があります。

## なぜ会社で大事なのか

会社では、AI の費用は個人だけの問題ではありません。営業部が提案書を作る、経営企画が Excel を分析する、人事が候補者情報をまとめる、情シスが定期チェックをする。こうした使い方が同じ credits pool から消費されるなら、誰がどれだけ使ったかを確認する必要があります。

[ChatGPTタスク刷新](/blog/openai-chatgpt-scheduled-tasks-management-2026/) でも見たように、ChatGPT はリマインダーや定期チェックの道具にも近づいています。[ChatGPT for Excel](/blog/openai-chatgpt-for-excel-financial-data-2026/) は表計算の作業面に入る話でした。今回の料金更新は、その便利さに費用管理が追いつく必要がある、という話です。

## 使う前に決めること

まず、どの仕事で使ってよいかを決めます。軽い表の整形と、財務モデルや顧客データの分析は同じではありません。社内向けの資料作成と、顧客へ出す提案書も同じではありません。

次に、credits を誰が見るかを決めます。ChatGPT の管理者だけでなく、経理、情シス、業務部門の責任者も関係します。月末に請求を見て驚くのではなく、最初は週ごとに利用量を見るのが安全です。

最後に、使わないエージェントを止める手順を決めます。毎日動くエージェントは、小さく見えても積み上がります。不要になったスケジュール、古い資料作成、使われていないエージェントを放置しないことが大切です。

## まとめ

今回の更新で、ChatGPT の業務AIは無料のお試しから、credits を使う仕事道具へ近づきました。Excel/Sheets と Workspace Agents はすでに課金対象で、PowerPoint も 2026年8月6日後に同じ方向へ進みます。

会社で使うなら、便利そうだから広げるのではなく、業務ごとの使い方、credits の責任者、無料期間中の実測、不要なエージェントの停止を先に決めましょう。
